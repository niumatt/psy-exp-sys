# 心理实验被试管理系统

一个面向高校心理学实验室的被试招募与管理平台，支持被试报名、主试发布实验、管理员审核认证等完整流程。

## 功能规划

### 被试端（微信小程序）
- 浏览招募中的实验列表
- 通过列表、扫描二维码或输入实验 ID 进入实验详情页
- 在详情页报名参与实验
- 支持跳转外部问卷小程序（腾讯问卷、问卷星）完成筛选问卷后报名
- "我的报名"页面：查看和管理已报名的实验

### 主试端（微信小程序 / Web，待定）
- 申请成为主试
- 发布、编辑实验
- 查看实验报名情况
- 审核被试资格

### 管理端
- 认证专业被试与主试身份
- 管理专业被试的实验学分记录

## 技术栈

- **前端**：微信小程序（原生）+ [TDesign MiniProgram](https://tdesign.tencent.com/miniprogram/)
- **后端**：微信云开发（云数据库 + 云函数）
- **外部集成**：腾讯问卷小程序、问卷星小程序

## 数据库集合

| 集合 | 说明 |
|------|------|
| `Experiments` | 实验信息 |
| `Subjects` | 被试信息 |
| `Appointments` | 报名记录 |
| `QuestionaryRecords` | 问卷记录 |
| `Slots` | 实验时间段 |
